# Welcome to my github.io website repository

Hope you have a lovely stay! This is my [personal website](justivanr.github.io)!

## Deploy your changes using Hexo

When you make changes to your website, start by running:

```bash
hexo clean
```

This removes any previously generated files — essentially giving you a clean slate.

Next, deploy your changes with:

```bash
hexo deploy
```

This pushes your updates to GitHub (note: a GitHub Action will run before the changes go live).

To preview your website locally, run:

```bash
hexo server
```

This will launch the site on your localhost.
